Semantic Propagation System

Founding Engineer home assignment — designing how a multi-tenant data platform safely corrects an incorrect semantic field definition and propagates that change across a live dependency graph.

The Incident

Field event_timestamp in a shared base dataset was documented as a free-text string but actually stores ISO 8601 dates (YYYY-MM-DD, UTC, no time component). This field flows into DAU metrics, cross-region aggregations, billing deduplication, ML feature tables, and identity joins — affecting millions of records across multiple tenants. Fixing the definition changes the meaning of historical data without touching the bytes on disk.
